# primus-daily

### 1. Clone Repository

```bash
git clone https://github.com/harunros66/primus.git
```

### 2. Go To Bot Directory
```
cd primus
```

### 3. Install
```
npm install axios fs path node-schedule
```

### 4. set up token bearer
```
nano token.txt
```

### 5. run

```bash
node primus --now
```
